📊 ייצוא וייבוא CSV (אקסל)

ייצא את רשימת האירועים שלך לקובץ אקסל לעריכה במחשב, או ייבא רשימה מוכנה מאקסל.
שימושי לעריכת הרבה אירועים בבת אחת.

איך מייבאים מאקסל?
1. לחץ "ייצוא ל-CSV" כדי לקבל קובץ לדוגמה
2. פתח אותו באקסל / Google Sheets
3. ערוך או הוסף שורות
4. שמור כ-CSV והעלה עם "ייבוא מ-CSV"
📅 ייבוא מלוח שנה (ICS)

ייצא קובץ .ics מגוגל קלנדר / אפל / אאוטלוק והעלה אותו כאן.
כל האירועים ייכנסו אוטומטית לאפליקציה.

איך מייצאים מגוגל קלנדר?
1. כנס ל-calendar.google.com במחשב
2. הגדרות ← ייצוא
3. הורד את קובץ ה-ZIP, פתח אותו
4. העלה כאן את הקובץ .ics
💾 שמירה ושחזור גיבוי

מחליף טלפון? שמור גיבוי בטלפון הישן ← שחזר בחדש — וכל המידע חוזר! 📱➡️📱
